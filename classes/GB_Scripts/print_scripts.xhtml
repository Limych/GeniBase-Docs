GB_Scripts::print_scripts

Prints scripts.

Prints the scripts passed to it or the print queue. Also prints all necessary dependencies.

    Signature

    public function print_scripts([boolean $handles = false, [boolean $group = false]] )

    Parameters

    $handles — boolean

    Optional. Scripts to be printed. (void) prints queue, (string) prints
    that script, (array of strings) prints those scripts. Default false.
    $group — boolean

    Optional. If scripts were queued in groups prints this group number.
    Default false.

    Returns

    array
    Scripts that have been printed.
