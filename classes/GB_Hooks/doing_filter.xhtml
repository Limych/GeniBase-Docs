GB_Hooks::doing_filter

Retrieve the name of a filter currently being processed.

The function GB_Hooks::current_filter() only returns the most recent filter or action being executed. GB_Hooks::did_action() returns true once the action is initially processed. This function allows detection for any filter currently being executed (despite not being the most recent filter to fire, in the case of hooks called from hook callbacks) to be verified.
