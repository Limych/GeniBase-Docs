GB_Hooks::has_action

Check if any action has been registered for a hook.

    Signature

    public function has_action(string $tag, [boolean $function_to_check = false] )

    Parameters

    $tag — string

    The name of the action hook.
    $function_to_check — boolean

    Optional. The callback to check for. Default false.

    Returns

    bool|int
    If $function_to_check is omitted, returns boolean for whether the hook has
    anything registered. When checking a specific function, the priority of that hook is returned, or false if the function is not attached. When using the $function_to_check argument, this function may return a non-boolean value that evaluates to false (e.g.) 0, so use the === operator for testing the return value.
