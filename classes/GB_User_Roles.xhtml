\GB_User_Roles

GeniBase User Roles.

The role option is simple, the structure is organized by role name that store
the name in value of the 'name' key. The capabilities are stored as an array
in the value of the 'capability' key.

array (
'rolename' => array (
'name' => 'rolename',
'capabilities' => array()
)
)
