GB_Options::set_transient

Set/update the value of a transient.

You do not need to serialize values. If the value needs to be serialized, then it will be serialized before it is set.

    Signature

    public function set_transient(string $transient, mixed $value, [integer $expiration = 0] )

    Parameters

    $transient — string

    Transient name. Expected to not be SQL-escaped. Must be
    45 characters or fewer in length.
    $value — mixed

    Transient value. Must be serializable if non-scalar.
    Expected to not be SQL-escaped.
    $expiration — integer

    Optional. Time until expiration in seconds. Default 0.

    Returns

    bool
    False if value was not set and true if value was set.
