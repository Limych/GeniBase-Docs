GB_Error::__construct

Initialize the error.

If `$code` is empty, the other parameters will be ignored. When `$code` is not empty, `$message` will be used even if it is empty. The `$data` parameter will be used only if it is not empty. Though the class is constructed with a single error code and message, multiple codes can be added using the `add()` method.
