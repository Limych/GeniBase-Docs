GB_User::sanitize_field

Sanitize user field based on context.

Possible context values ($this->filter) are: 'raw', 'edit', 'db', 'display', 'attribute' and 'js'. The 'display' context is used by default. 'attribute' and 'js' contexts are treated like 'display' when calling filters.
